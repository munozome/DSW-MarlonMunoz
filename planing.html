    primera seccion

    Generalidades:

    Para aplicar las hojas de estilo en cascada se hace necesario entender las 3 formas de aplicarlo, las cuales son: Locales, internas y externas.

    Locales

    Estas se hacen dentro de las etiquetas en el código html, usando el atributo style, por ejemplo si estamos escribiendo una etiqueta b usada para la negrita dentro de esta ponemos style = y entre comillas el atributo por ejemplo color y luego de cerrada la etiqueta escribiriamos el atributo (imagen)

    Internas

    Para esto los selectores y atributos se colocarian dentro de un style en la parte head del documento html (Imagen)

    Externas

    En este consistiría en hacer un archivo con extensión css, en este se escribirá el código de los estilos y para enlazar dicha hoja se coloca en el head también pero a diferencia de que se hará con un link rel que nos permitirá llamar la hoja de estilos, no hay que olvidar colocar el nombre correcto del archivo css que se creo

    Se relacionará también algo de la historia de este lenguaje


    segunda seccion

    Selectores

    En esta sección se hablará un poco de los selectores básicos que se utilizan los cuales son:

    Universales:

    Aplica los estilos a todas las etiquetas, normalmente se usan con margin y padding

    etiqueta

    Se aplican los estilos a las etiquetas que concidan con el valor descrito en el selector

    descedente:

    Se aplican a los elementos incluidos entre otro

    Clases

    Se apkican a los que estan incluidos dentro de una clase creada, y se le agrega un punto antes del nombre del selector el cual es el nombre de la clase

    ID

    Se aplican a elementos unicos que tienen asignados un id, igualmente el nobre del selector será el nombre del id, en este caso se agregara # antes del mismo
