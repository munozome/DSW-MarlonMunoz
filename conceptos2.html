Selectores y estilos con CSS 3


Gracias a los selectores cuando creamos una página web en html y una hoja de estilos ya sea externa o interna podemos unirlas para que se apliquen los estilos definidos en la hoja seleccionada. Tenemos estilos aplicables a tablas, contenedores, bloques, fuente, color, fondos entre otros que podremos ver más adelante. Sobre todo no hay que olvidar que la regla de estilo se conforma del selector y la declaración. Le llamamos declaración al conjuntos de pares propiedad/valor que se incluyen en los estilos. Tenemos varios tipos de selectores los cuales trataremos a continuación:




Selector Universal:

Aplica los estilos a todas las etiquetas, normalmente se usan con margin y padding



Selector Etiqueta:

Se aplican los estilos a las etiquetas que concidan con el nombre del selector




Selector Clase:

Este lleva el nombre de la clase a la cual se quiere aplicar y se le agrega un punto antes del nombre del selector


Selector Identificador ID:

Se aplican a elementos únicos que tienen asignados un id, igualmente el nombre del selector será el nombre del id y se le agrega al inicio del mismo un #

Selector descendiente:

Son aquellos que se encuentran en las etiquetas de apertura y cierre del "padre"


Otros selectores:

Tenemos los selectores hijos que se representan con un mayor que (>) y cuando tomamos elementos de un mismo padre usamos (+) entre los nombres de los selectores que serían hermanos, conocidos como selectores adyacentes

Estilos

Como lo mencionabamos anteriormente hay estilos aplicables al color, fuente, bordes, etc, a continuación trabajaremos los más importantes a la hora de trabajar estilos.

Color Fuentes Texto
Para aplicar colores tenemos 3 formas que son las mas utilizadas.
La primera es:

RGB:

Con esta lo que hacemos es asignar el color mencionando que cantidad de rojo (red), verde (green) y azul (blue) tiene el color que deseamos aplicar 		Las fuentes se asignan gracias a la etiqueta font-family, es gracias a una familia de fuentes que se conforma un tipo de letra determinado, aunque también los distintos valoes que tiene la etiqueta mencionada permite que si para el navegador alguna fuente es desconocida pueda tener otras opciones Gracias a la etiqueta text-align podemos centrar un texto, justificarlo, alinearlo a la izquierda y a la derecha

HEX:

El método hexadecimal permite asignar un color a determinado selector con el número que le a sido asignado este color se conforma de un# y 8 números, el primer par es la cantidad del color rojo, el segundo del color verde y el ultimo del color azul 		A una fuente podemos aplicarle un color determinado y estilo, para el color usamos la etiqueta color y para el estilo la etiqueta font-style en esta última podemos poner en negrita, en italica, obliqua, normal, etc Gracias a la hoja de estilo podemos colocar decoración al texto con text-decoration.

Nombre del color:

A través de internet podemos buscar los nombres de los colores que estan disponibles en htm estos se escriben en inglés, ejemplo: black, white, cyan, blue, etc 		Podemos ajustar el tamaño de la fuente con la propiedad font-size que nos permitirá cambiar el tamaño que trae por defecto en el html tanto en los textos con formato (p) que son utilizados para los párrafos, asi como los h1, h2, h3 h4, etc Cuando queremos ahorrarnos tiempo a la hora de escribir el código en html, pero queremos mayusculas y minusculas, podemos escribir todo en minúscula y a la hora de aplicar estilos con un text-transform podemos poner todo en mayúscula, minuscula, o mayúscula inicial.
