¿Qué es CSS3?

CSS (Cascading Style Sheets) es un lenguaje de marcado que nos permite dar estilo a una página en la cual estemos trabajando. Existen 3 versiones actualmente que son la css1 css2 y la css3, es necesario tener en cuenta que las dos primeras son totalmente interpretadas por los navegadores y la última faltan cosas por mejorar en los motores de búsqueda. Dependiendo de donde se ejecute el html que tiene el css 3 aplicado podremos ver diferencias pues en ocasiones el navegador podría no leer los estilos que se aplican cuando realizamos acciones con el mouse o no entiende algunas propiedades.



¿Quién lo creó?

Las hojas de estilo en cascada fueron creadas por Hakon Wium Lie y Bert Bos, hacen su primera presentación conjunta sobre hojas de estilo el 14 de abril de 1995 en la conferencia de World Wide Web en Darmstadt, Alemania; en ese momento Bos demostró la implementación de hojas de estilo en Argo que era el navegador que en ese tiempo estaba diseñando Bert. En la primera versión el css solo ayudaba a colocar imagenes, espacios, margenes, cambiar aspectos enel tipo de letra y el color, ahora tenemos más cosas las cuales destacamos.

  • Usar colores en degrade o gradientes
  • Redondear los bordes de las tablas
  • También permite configurar la página de modo que se adapte a cualquier dispositivo



    • ¿Cómo aplico hojas de estilo en cascada?

    Hay 3 formas de aplicar una hoja en cascada (CSS), sin embargo cuando desarrollamos una página web podemos utilizar las 3 formas, no obstante es necesario tener en cuenta que el navegador leera primero los estilos locales, luego los internos y por último las hojas de estilo externas, este último es el más aconsejable

    Locales:

    Estas se hacen dentro de las etiquetas en el código html, usando el atributo style, por ejemplo si estamos escribiendo una etiqueta b usada para la negrita dentro de esta ponemos style = y entre comillas el atributo por ejemplo color y luego de cerrada la etiqueta escribiriamos la información a la que aplicamos el estilo

    Interna:

    Para esto los selectores y atributos se colocarian en la etiqueta head del documento html con una etiqueta style. Es de tener en cuenta que los estilos se componen del selector y la declaración esta última haría referencia a las propiedades con su respectivo valor

    Externa:

    Este consiste en hacer un archivo con extensión css, en este se escribirá el código de los estilos y para enlazar dicha hoja se coloca en el head también pero a diferencia de que se hará con un link que nos permitirá llamar la hoja de estilos, no hay que olvidar colocar el nombre correcto del archivo css que se creó
